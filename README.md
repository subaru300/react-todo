# React App Todo List
![screenshot 1](https://github.com/subaru300/react-todo/blob/main/screenshot1.png)

## I used
- ReactJS
- Chakra UI
- Framer Motion

## Features
- You can create task, check it done and delete. 
- All tasks saves automatically to localStorage
- You can change dark/light theme
